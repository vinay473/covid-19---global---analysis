# 🌍 COVID-19 Global Data Analysis & Visualization

This project focuses on analyzing and visualizing the global impact of the COVID-19 pandemic using real-world data from [Our World in Data](https://ourworldindata.org/coronavirus-source-data).

## 📊 Features
- Total confirmed cases, deaths, and recoveries
- Daily and cumulative trends across countries
- Mortality and recovery rate calculations
- Top 10 most affected countries
- Visualizations using Python (Matplotlib, Seaborn)

## 🧰 Tech Stack
- Python 3
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook
